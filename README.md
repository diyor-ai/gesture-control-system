# Gesture Control System

Advanced hand gesture-based computer control system using MediaPipe and OpenCV.

## Features
- Real-time mouse movement and click
- Volume Control
- Screen Scrolling
- Finger Drawing
- Screenshot Capture
- Window Movement
- Zoom In/Out
- Media Playback Control

## Technologies
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## How to Run
1. Clone the repo
2. Install dependencies
3. Run `main.py`
